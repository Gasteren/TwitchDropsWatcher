# Twitch Drops Watcher

A World of Warcraft addon to track active Twitch Drop campaigns in-game.

---

## Features

- Live countdown timers per campaign
- Reward details, requirements, and icons
- Ctrl+Click the icon to preview the reward in the Dressing Room
- "I have this drop" checkbox to stop notifications for drops you already own
- Login notifications with optional sound alert
- Minimap button for quick access
- Dark mode UI — movable, scrollable, closes with ESC
- Settings panel built-in, no extra dependencies needed

## Install

Download from [CurseForge](https://www.curseforge.com/wow/addons/twitch-drops-watcher)

All libraries are bundled — nothing else to install.

---

## Commands

| Command | Action |
|---------|--------|
| `/tdw` | Open/close the main window |
| `/tdws` | Open/close settings |

---

## Settings

Accessible via the gear icon in the header.

| Option | Description |
|--------|-------------|
| Notify on Login | Print active campaigns to chat on login |
| Play Sound | Play a sound when campaigns are found |
| Auto-Open UI | Open the window automatically on login |
